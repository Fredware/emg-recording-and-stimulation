# emg-recording-and-stimulation
The code in this repository is for the Jedi Scientist Demo developed by the [NeuroRobotics Lab](https://neurorobotics.ece.utah.edu/research/) at the University of Utah.
The code lets users visualize their surface EMG and stimulate muscles through skin electrodes.
